## 세션 시작 시 필독

이 프로젝트는 **Claude와 Codex 등 다른 ai프로그램을 번갈아 사용**할 수 있습니다.
세션 시작 시 다음 파일들을 반드시 읽고,
사용자 요청 작업을 수행하기 전에
현재 프로젝트 상태를 5줄 이내로 요약하세요.
작업 시작 전:

1. CLAUDE.md 확인
2. HANDOFF.md 확인
3. TODO.md 확인
4. 현재 Git 상태 확인
5. 작업 계획 제시

순서로 진행한다.

1. `.ai-handoff/HANDOFF.md` - 프로젝트 현황 및 구조
2. `.ai-handoff/TODO.md` - 현재 할 일 목록
3. `.ai-handoff/DECISIONS.md` - 기술적 결정 및 주의사항
4. `.ai-handoff/SESSION_LOG.md` - 최신 세션 로그

TODO.md에 우선순위(P0/P1/P2)가 있다면
반드시 가장 높은 우선순위부터 처리한다.
