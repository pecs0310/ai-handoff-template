## 🔵 기본 시작 프롬프트 (공통 / 매 세션 시작 시 사용)

```

작업을 시작하기 전에 반드시 .ai-handoff/HANDOFF.md, .ai-handoff/TODO.md,
.ai-handoff/DECISIONS.md, .ai-handoff/SESSION_LOG.md를 먼저 읽고
현재 목표, 진행 상황, 주의사항을 파악해 주세요.

작업 중에는 사용자가 직접 만든 변경사항을 되돌리지 말고,
관련 없는 리팩터링은 피하고, 기존 코드 스타일과 구조를 우선해 주세요.

작업이 끝나면 .ai-handoff/ 문서들을 갱신해 주세요.
이제 인계 파일들을 읽고 이어서 작업해 주세요.
```
